## 体会一：不同键值数据库的数据类型差异较大

键值数据库属于后端系统，因此我们在选择键值数据库的时候，**从业务应用的角度来看，首先就会考虑键值数据库能提供的数据类型有哪些**。而对于Redis和其他键值数据库来说，虽然我们都称之为键值数据库，它们在保存数据时，实际也是按照键-值的方式来保存的，但是，它们呈现出来的数据类型还是有差别的。
