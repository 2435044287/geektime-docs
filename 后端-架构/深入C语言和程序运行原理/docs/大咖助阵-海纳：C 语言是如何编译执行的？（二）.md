## 文法分析

文法，或者叫语法（Grammar），它描述了一套语言的产生规则。例如，一个合法的句子包含主语、谓语和宾语。那么，我们就可以这样定义句子的规则：

$Sent\\rightarrow S P O$

其中，Sent代表一个句子，S代表主语（Subject），P代表谓语（Predicate），O代表宾语（Object）。上述公式可以这么理解：句子可以**推导**成主语加谓语加宾语的结构。
