## 基础知识

那所谓的基础知识是什么呢？**我认为能够满足开发业务的最小知识集合都是基础。**比如，语言、框架和一些编程的必备工具，还有 React Native 框架和社区提供的核心组件、核心 API，再有就是熟悉工作流包括搭建环境、开发流程、上线流程，这些都是 React Native 的基础知识。
