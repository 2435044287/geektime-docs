## 不要过早做性能优化

著名计算机科学家**高德纳**（Donald E. Knuth）在他的著作《计算机程序设计艺术》（TAOCP）中提出过：“**过早优化是万恶之源（Premature Optimization is the Root of All Evil）**。”强调了开发者在程序开发的早期阶段，有许多重要的工作要做，而优化程序这件事往往会占用了开发者过多的精力，却没能带来对等的收益。

当你在开发前端应用时，也可以参考这一原则。
